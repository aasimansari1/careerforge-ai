## ⚡ Overview

**CareerForge AI** is a full-featured AI-powered resume builder with 30+ career tools and 40+ ATS-optimized templates. Generate, customize, and export professional resumes in minutes using AI assistance.

---

## 🚀 Features

| Category | Features |
|:---|:---|
| **AI Tools** | AI resume writer, cover letter generator, skill gap analyzer |
| **Templates** | 40+ ATS-optimized, industry-specific templates |
| **Editor** | Drag-and-drop sections, real-time preview, inline editing |
| **Export** | PDF, DOCX, and shareable link |
| **Analysis** | ATS score checker, keyword optimizer |
| **Extras** | LinkedIn import, job description matcher |

## ⚙️ Quick Start

```bash
# 1. Clone
git clone https://github.com/aasimansari1/careerforge-ai.git
cd careerforge-ai

# 2. Install
npm install

# 3. Environment
cp .env.example .env.local
# Add: OPENAI_API_KEY, NEXTAUTH_SECRET, MONGODB_URI

# 4. Run
npm run dev
```

Open **http://localhost:3000**
